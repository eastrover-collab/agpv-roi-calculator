# 영농형 태양광 경제성 계산기

사단법인 식량과기후(KIFC)가 운영하는 Streamlit 기반 의사결정 보조 도구입니다. WordPress의 계산기 소개 페이지에 임베드되며, 계산 앱 자체는 Streamlit Community Cloud에서 별도로 배포됩니다.

## v2 계산 원칙

- 사업 현금흐름: 총사업비를 Year 0에 전액 반영하고 금융 조건을 제외합니다.
- 자기자본 현금흐름: 초기 자기자본과 연도별 실제 원리금 상환을 반영합니다.
- 상환 여력: 원리금이 있는 연도의 DSCR 최솟값을 제공합니다.
- 인버터: 연간 적립액의 10배를 10년차와 20년차에 일시 반영합니다.
- 요구수익률은 대출금리와 분리합니다.

세금·부가가치세, 임차료, 계통 보강비와 출력제어, 철거·원상복구비, 물가상승은 현재 포함하지 않습니다. 2026년 법률 통과 내용은 분석기간 가정에 반영하되, 실제 허가와 지원 여부는 시행령·공고·개별 심사를 확인해야 합니다.

## 실행과 검증

```bash
pip install -r requirements.txt
streamlit run app.py
pytest -q
```

`main` 브랜치에 푸시하면 연결된 Streamlit Cloud 앱이 다시 배포됩니다. 저장소 공개 여부와 별개로 Streamlit 앱의 공유 설정이 비공개이면 익명 사용자는 로그인 화면으로 이동하므로, 배포 후 익명 창에서 공개 접근을 반드시 확인해야 합니다.

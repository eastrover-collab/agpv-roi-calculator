# 영농형 태양광 경제성 계산기 사용 안내

이 계산기는 초기 검토용입니다. 농지 면적, 총사업비, 자기자본 비율, 전력 판매단가와 대출 조건을 바꾸면 결과가 즉시 갱신됩니다.

## 결과를 읽는 순서

1. `사업 순현재가치`와 `사업 내부수익률`로 설비 자체의 경제성을 봅니다.
2. `자기자본 내부수익률`과 회수기간으로 농가가 실제 투입한 돈의 흐름을 봅니다.
3. `최저 DSCR`이 1보다 작은지 확인합니다. 1 미만이면 일부 연도에 영업현금만으로 원리금을 갚기 어렵다는 뜻입니다.
4. 현금흐름 탭에서 원금상환 시작 연도와 인버터 교체 연도의 현금 부족을 확인합니다.
5. 민감도 탭에서 판매단가 하락과 설치비·금리 상승 조건을 확인합니다.

## 주의할 가정

기본값은 전남, 논벼, 2,000㎡, 99kW의 단순화된 사례입니다. 용량과 비용은 실제 설계·계통 조건에 따라 달라집니다. 세금·부가가치세, 임차료, 계통 보강비와 출력제어, 철거·원상복구비, 물가상승은 계산에 포함되지 않습니다.

2026년 법률 통과 내용은 23년 분석기간 가정에 반영했습니다. 다만 실제 시행일, 대상지역, 영농 지속 기준, 허가기간과 금융지원 요건은 하위법령과 최신 사업 공고를 확인해야 합니다.

## 외부 서비스

계산 앱은 Streamlit Community Cloud에서 제공됩니다. WordPress 소개 페이지가 열리더라도 외부 앱의 장애·절전·공개 설정에 따라 계산기가 표시되지 않을 수 있습니다. 이때는 소개 페이지의 `새 창에서 계산기 열기`를 사용하고, 로그인 화면이 나오면 KIFC에 알려 주세요.

# 영농형 태양광 경제성 계산기 사용 가이드

농지 위에서 작물을 계속 재배하면서 태양광 발전 수익을 함께 얻을 수 있을까. 이 계산기는 농가·상담자·정책 담당자가 같은 질문을 같은 숫자로 검토할 수 있도록 만든 의사결정 도구입니다.

계산기는 KREI(2023) 「영농형 태양광 도입의 경제성 분석」 4장 모델을 기반으로 하되, 2026년 5월 기준 시장 데이터와 농지법 개정에 맞춰 기본값을 갱신했습니다. 최근 버전에서는 간편/전문가 입력 모드, 용량별 운영비 보정, 현재 입력값 기준 시나리오, 연도별 발전량 감소를 반영한 NPV 계산, PDF 요약 보고서 다운로드를 추가했습니다.

계산기 바로가기: https://agpv-roi-calculator-2000m.streamlit.app/

---

## 1. 이 계산기로 알 수 있는 것

계산기는 사용자가 입력한 농지·시설·금융·전력판매 조건을 바탕으로 다음 지표를 산출합니다.

| 지표 | 의미 | 해석 기준 |
|---|---|---|
| B/C | 현재가치 기준 편익 ÷ 비용 | 1보다 크면 편익이 비용보다 큼 |
| NPV | 사업 전체 순현재가치 | 0보다 크면 할인 후에도 순이익 |
| IRR | 내부수익률 | 조달금리·요구수익률보다 높을수록 유리 |
| 회수기간 | 초기 자기자본을 회수하는 시점 | 짧을수록 유동성 부담이 낮음 |
| 월별 통장 흐름 | 실제 월 단위 현금 유입·유출 | 거치 종료 후 상환 부담 확인 |
| 시나리오 | 가격·설치비·금리 변화별 수익성 | 어떤 변수에 취약한지 확인 |

이 계산기는 최종 투자 결정을 대신하지 않습니다. 계통연계 가능 여부, 인허가, 실제 견적, 작물별 생육 영향, 세무 조건은 반드시 별도로 확인해야 합니다.

## 2. 빠른 사용법

### 2.1 간편 모드

처음 사용하는 농가라면 간편 모드를 권장합니다.

1. 계산기 왼쪽 사이드바에서 입력 모드를 `간편`으로 선택합니다.
2. 농지 면적을 입력합니다.
3. 지역은 현재 `전라남도` 기준입니다.
4. 자기자본 비율과 융자 조건을 선택합니다.
5. 발전가격 방식은 `변동형 판매가격` 또는 `고정가격계약` 중 선택합니다.
6. 오른쪽의 `한눈에 보기` 탭에서 B/C, 회수기간, IRR, 연간 순이익을 확인합니다.
7. 상담 자료가 필요하면 `요약 보고서 PDF 받기`를 눌러 현재 입력값 기준 보고서를 저장합니다.

간편 모드에서는 계산기가 다음 값을 자동 추천합니다.

| 항목 | 자동 적용 방식 |
|---|---|
| 시설 용량 | 2,000㎡당 99kW 기준으로 면적 비례 추천 |
| 발전시간 | 전남 기준 1일 평균 3.7h |
| 총 사업비 | 99kW 2.1억원 기준으로 용량 비례 추천 |
| 발전단가 | RPS 또는 PPA 기본값 적용 |
| 운영비 | 시설 용량과 총사업비에 맞춰 자동 보정 |

### 2.2 전문가 모드

상담자, 연구자, 시공사, 정책 담당자는 전문가 모드를 사용하면 세부 가정을 직접 바꿀 수 있습니다.

전문가 모드에서 조정 가능한 항목은 다음과 같습니다.

| 구분 | 조정 항목 |
|---|---|
| 시설 | 시설 용량, 1일 평균 발전시간 |
| 사업비 | 총 사업비 |
| 금융 | 자기자본 비율, 융자 조건, 직접 금리 입력 |
| 발전가격 | RPS/PPA 선택, SMP, REC, REC 가중치, PPA 단가 |
| 결과 공유 | 현재 입력값이 포함된 공유 URL |

전문가 모드는 실제 견적서, 금융 상담 결과, 전력판매 조건을 알고 있을 때 사용하면 좋습니다.

## 3. 탭별 읽는 법

### 3.1 한눈에 보기

가장 먼저 볼 화면입니다.

- `종합 평가`: B/C 기준으로 수익성 상태를 신호등처럼 표시합니다.
- `연간 순이익`: 벼와 태양광을 합산한 연간 환산 순이익입니다.
- `투자 회수 기간`: 초기 자기자본이 누적 현금흐름으로 회수되는 시점입니다.
- `내부수익률(IRR)`: 사업 현금흐름이 만드는 수익률입니다.
- `B/C 비율`: 현재가치 기준 편익과 비용의 비율입니다.

해석 기준:

| B/C | 판정 |
|---|---|
| 1.2 이상 | 수익성 있음 |
| 1.0 이상 1.2 미만 | 경계, 조건 변화 확인 필요 |
| 1.0 미만 | 손실 우려 |

### 3.2 월별 통장 흐름

B/C는 사업 전체의 현재가치 지표입니다. 그러나 농가가 실제로 체감하는 것은 매월 통장에 들어오고 나가는 돈입니다.

이 탭은 다음 구간을 나눠 보여줍니다.

| 구간 | 의미 |
|---|---|
| Year 0 | 자기자본 일시 투입 |
| Year 1~5 | 거치기간, 이자만 납부 |
| Year 6~15 | 원금+이자 상환 |
| Year 16 이후 | 융자 완료 후 현금흐름 개선 |
| Year 10, 20 | 인버터 교체 일시 비용 반영 |

특히 6년차부터 원금 상환이 시작되면 월별 현금흐름이 나빠질 수 있습니다. B/C가 양호해도 중간 상환기에 현금 부족이 생길 수 있으므로 이 탭을 반드시 확인해야 합니다.

### 3.3 23년 시뮬레이션

사업 시작 시점부터 23년간 누적 현금흐름을 보여줍니다.

- Year 0은 초기 자기자본 지출로 음수에서 시작합니다.
- 누적선이 0을 넘는 시점이 투자 회수 시점입니다.
- 연도별 수입·지출 막대그래프로 특정 연도 비용 부담을 확인할 수 있습니다.

### 3.4 시나리오 비교

최근 버전부터 시나리오는 고정된 PDF 예시값이 아니라 현재 입력값을 기준으로 생성됩니다.

비교 항목:

| 시나리오 | 설명 |
|---|---|
| 현재 입력값 | 사용자가 입력한 기준 조건 |
| 발전단가 -15% | SMP/REC 또는 PPA 가격 하락 |
| 발전단가 +15% | 발전판매가격 상승 |
| 설치비 -15% | 견적 절감 또는 보조금 효과 |
| 설치비 +15% | 공사비 증가 |
| 금리 -1%p | 정책금리 인하 또는 저리 융자 |
| 금리 +1%p | 금리 상승 |

이 탭은 “어떤 변수가 사업성을 가장 많이 흔드는가”를 빠르게 확인하는 데 유용합니다.

### 3.5 리스크 점검

발전가격, 설치비, 금리를 흔들었을 때 B/C가 얼마나 바뀌는지 토네이도 차트로 보여줍니다. 막대가 길수록 해당 변수에 취약합니다.

또한 손익분기 임계값을 제공합니다.

- 발전가격이 얼마 이하로 내려가면 B/C 1.0 미만이 되는가
- 설치비가 현재 대비 몇 %까지 증가하면 손익분기 이하가 되는가

### 3.6 전문가 모드 탭

상세 재무 지표, 복합 시나리오, 리스크 프리미엄, 농지법 개정 효과를 확인하는 탭입니다.

이 탭에는 KREI(2023) 표 재현용 시나리오도 남겨두었습니다. 일반 사용자용 시나리오 탭은 현재 입력값 기준이고, 전문가 모드는 원 보고서와 비교하기 위한 고정 시나리오를 포함합니다.

### 3.7 PDF 요약 보고서

화면 상단의 `요약 보고서 PDF 받기` 버튼을 누르면 현재 입력값 기준의 보고서를 저장할 수 있습니다.

PDF에는 다음 내용이 포함됩니다.

| 구분 | 포함 내용 |
|---|---|
| 기관 정보 | 사단법인 식량과기후 공식 로고 |
| 종합 평가 | B/C, IRR, 회수기간, 연간 순이익 |
| 입력 조건 | 면적, 용량, 사업비, 자기자본, 금리, 발전가격 |
| 연간 수입·지출 | 발전수익, 작물수익, 금융비용, 주요 운영비 |
| 현재가치 지표 | 태양광 NPV, 작물소득 현재가치, 총 NPV, 할인율 |
| 공유 정보 | 같은 입력값을 다시 열 수 있는 공유 링크 |

농가 상담, 내부 검토, 지자체 협의처럼 계산 화면을 그대로 함께 보기 어려운 상황에서 사용하면 좋습니다.

## 5. 계산 수식

### 5.1 시설 용량 추천

간편 모드의 추천 용량은 면적에 비례합니다.

```text
추천 용량(kW) = 농지면적(㎡) × 99 / 2,000
```

예: 2,000㎡이면 99kW, 4,000㎡이면 약 198kW입니다.

### 5.2 연도별 발전량

최근 버전부터 NPV 계산에도 연도별 발전효율 감소를 직접 반영합니다.

```text
1년차 발전량(kWh) = 시설용량(kW) × 1일 평균 발전시간(h) × 365

t년차 발전량(kWh)
= 1년차 발전량 × (1 - 발전효율 감소율)^(t - 1)
```

기본 감소율은 연 1.1%입니다.

### 5.3 평균 연간 발전량

화면의 연간 평균 발전량은 운영기간 동안의 평균값입니다.

```text
평균 연간 발전량
= [Σ t=1..n {1년차 발전량 × (1 - d)^(t - 1)}] / n
```

여기서 `d`는 연 발전효율 감소율, `n`은 운영기간입니다.

### 5.4 발전단가

RPS 트랙:

```text
발전단가(원/kWh) = SMP + REC × REC 가중치
```

기본값:

```text
109.6 + 71.3 × 1.2 = 195.16원/kWh
```

PPA 트랙:

```text
발전단가(원/kWh) = 고정가격계약 단가
```

기본값은 154.7원/kWh입니다.

### 5.5 연간 발전수익

```text
t년차 발전수익(천원)
= t년차 발전량(kWh) × 발전단가(원/kWh) / 1,000
```

원 단위 수익을 천원 단위로 바꾸기 위해 1,000으로 나눕니다.

### 5.6 작물 소득

```text
벼 기준소득(천원/년)
= 1,388 × 농지면적 / 2,000

영농형 하부 벼 소득
= 벼 기준소득 × (1 - 단수 감소율)
```

기본 단수 감소율은 20%입니다.

예:

```text
2,000㎡ 기준 벼 기준소득 = 1,388천원/년
영농형 하부 벼 소득 = 1,388 × (1 - 0.20) = 1,110.4천원/년
```

### 5.7 사업비

간편 모드의 추천 사업비는 현재 버전에서는 용량 비례로 계산합니다.

```text
추천 총사업비(천원)
= 기준 총사업비 × 시설용량 / 기준 시설용량
```

기본값:

```text
210,000 × 시설용량 / 99
```

향후 버전에서는 시설이 커질수록 kW당 단가가 낮아지는 규모의 경제를 반영할 수 있습니다.

### 5.8 운영비 자동 보정

최근 버전부터 운영비는 99kW 고정값을 그대로 쓰지 않고, 시설 용량과 총사업비에 맞춰 자동 보정합니다.

기본 운영비:

| 항목 | 99kW 기준 |
|---|---:|
| 인버터 교체비 연간분할 | 1,000천원/년 |
| 전기안전관리대행 | 1,200천원/년 |
| 보험료 | 540천원/년 |
| 폐기물 처리 | 250천원/년 |
| 전기료·수선비 | 420천원/년 |

보정 방식:

```text
용량비율 = 실제 시설용량 / 기준 시설용량
사업비비율 = 실제 총사업비 / 기준 총사업비

인버터 교체비 = 기준 인버터 교체비 × 용량비율
폐기물 처리비 = 기준 폐기물 처리비 × 용량비율
전기료·수선비 = 기준 전기료·수선비 × 용량비율
보험료 = 기준 보험료 × 사업비비율
전기안전관리대행 = 기준 전기안전관리대행 × (0.7 + 0.3 × 용량비율)
```

전기안전관리대행은 완전 비례가 아니라 고정비 성격이 있으므로 70% 고정, 30% 용량비례로 처리합니다.

### 5.9 대출 상환

기본 금융 구조는 5년 거치, 10년 원금균등 상환입니다.

```text
대출원금 = 총사업비 × (1 - 자기자본비율)

거치기간 이자 = 대출잔액 × 금리

상환기간 연간 원금상환액 = 대출원금 / 분할상환기간

상환기간 연간 이자 = 해당 연도 초 대출잔액 × 금리
```

월별 통장 흐름 탭은 이 실제 상환 스케줄을 사용합니다.

### 5.10 NPV

NPV는 미래 현금흐름을 현재가치로 할인한 값입니다.

```text
PV = Σ t=1..n {현금흐름_t / (1 + 할인율)^t}
```

계산기의 기본 할인율은 입력한 융자 금리를 사용합니다.

총 편익 현재가치:

```text
PV 편익 = PV(발전수익 + 작물수익)
```

총 비용 현재가치:

```text
PV 비용 = 초기 자기자본 + PV(연간 운영비 + 대출상환)
```

순현재가치:

```text
NPV = PV 편익 - PV 비용
```

### 5.11 B/C

```text
B/C = PV 편익 / PV 비용
```

해석:

```text
B/C > 1.0  → 현재가치 기준 편익이 비용보다 큼
B/C = 1.0  → 손익분기
B/C < 1.0  → 비용이 편익보다 큼
```

계산기에서는 농가 의사결정용으로 B/C 1.2 이상을 양호 기준으로 표시합니다.

### 5.12 IRR

IRR은 NPV가 0이 되게 하는 할인율입니다.

```text
0 = Σ t=0..n {현금흐름_t / (1 + IRR)^t}
```

초기 자기자본은 Year 0의 음수 현금흐름으로 들어가고, 이후 연도별 순현금흐름으로 회수 여부를 계산합니다.

### 5.13 회수기간

회수기간은 누적 현금흐름이 0을 넘는 시점입니다.

```text
누적현금흐름_t = Σ Year 0..t 현금흐름
```

누적현금흐름이 음수에서 양수로 넘어가는 연도 사이를 선형 보간해 소수점 연도로 표시합니다.

## 6. 최근 변경사항

### 6.1 연도별 발전량 감소를 NPV에 반영

이전 방식은 운영기간 평균 발전수익을 매년 반복해 NPV를 계산했습니다. 최근 버전부터는 월별 현금흐름과 동일하게 매년 감소하는 발전량을 NPV에도 반영합니다.

변경 효과:

- 1년차 발전량이 가장 높고 이후 매년 조금씩 감소합니다.
- 장기 사업의 현재가치 계산이 더 일관됩니다.
- 월별 흐름, 23년 시뮬레이션, B/C 계산의 세계관이 맞춰졌습니다.

### 6.2 운영비 자동 보정

시설 용량이 달라져도 99kW 기준 운영비가 고정되는 문제를 개선했습니다.

이제 인버터, 폐기물, 수선비는 용량에 따라 변하고, 보험료는 총사업비에 따라 변합니다. 전기안전관리비는 고정비와 용량비례분을 섞어 반영합니다.

### 6.3 현재 입력값 기준 시나리오

일반 시나리오 탭은 더 이상 KREI PDF 고정값만 비교하지 않습니다.

사용자가 입력한 현재 조건을 기준으로 발전가격 ±15%, 설치비 ±15%, 금리 ±1%p를 비교합니다. 따라서 실제 상담 상황에서 “지금 이 농지·이 견적·이 금리라면 무엇이 위험한가”를 바로 볼 수 있습니다.

### 6.4 간편/전문가 모드 분리

처음 쓰는 사용자는 간편 모드에서 최소 입력만으로 결과를 볼 수 있습니다. 세부 가정을 알고 있는 사용자는 전문가 모드에서 가격·금리·사업비·발전시간을 직접 조정할 수 있습니다.

### 6.5 탭과 모드 선택 가독성 개선

상단 결과 탭과 사이드바의 간편/전문가 선택을 더 크게 표시해 모바일과 임베드 환경에서도 찾기 쉽게 개선했습니다.

### 6.6 PDF 요약 보고서 추가

현재 입력값을 기준으로 핵심 지표와 입력 조건, 연간 수입·지출, 현재가치 지표를 정리한 PDF를 다운로드할 수 있습니다. 보고서 하단에는 사단법인 식량과기후 공식 로고가 작게 들어가며, 공유 링크도 함께 담기므로 같은 조건을 다시 열어 후속 상담을 이어갈 수 있습니다.

## 7. 결과를 해석할 때 주의할 점

### 7.1 B/C가 좋아도 월별 현금흐름은 나쁠 수 있습니다

거치기간에는 이자만 내기 때문에 현금흐름이 좋아 보일 수 있습니다. 그러나 6년차부터 원금 상환이 시작되면 월별 흐름이 악화될 수 있습니다. 반드시 월별 통장 흐름 탭을 확인해야 합니다.

### 7.2 계통연계 가능 여부는 별도 확인해야 합니다

계산기는 경제성 모델입니다. 한전 계통연계 가능 여부, 변전소 여유용량, 접속 대기 기간은 별도 확인이 필요합니다. 계통 접속이 늦어지면 실제 발전수익 발생 시점이 밀려 사업성이 낮아질 수 있습니다.

### 7.3 작물별 결과는 아직 논벼 중심입니다

현재 기본 작물은 논벼입니다. 감자, 콩, 양파, 배추 등은 향후 확장 대상입니다. 작물이 달라지면 하부 생산성, 소득, 노동 투입, 시설 높이와 간격이 달라질 수 있습니다.

### 7.4 실제 견적은 반드시 반영해야 합니다

간편 모드의 총사업비는 추천값입니다. 실제 사업 검토 단계에서는 시공 견적, 계통연계비, 토목비, 인허가비, 보험료, 유지보수 계약 조건을 반영해야 합니다.

### 7.5 발전가격은 변동성이 큽니다

RPS 트랙은 SMP와 REC 가격에 영향을 받습니다. PPA 트랙은 고정가격 안정성이 있지만 계약 단가와 기간, 신용위험을 확인해야 합니다.

## 8. 상담 현장에서 권장하는 사용 순서

1. 간편 모드에서 농지 면적만 넣고 대략적인 수익성을 확인합니다.
2. `한눈에 보기`에서 B/C, 회수기간, IRR을 확인합니다.
3. `월별 통장 흐름`에서 6~15년차 상환 부담을 확인합니다.
4. 실제 견적이 있으면 전문가 모드로 전환해 총사업비를 수정합니다.
5. 계약 단가가 있으면 PPA 단가 또는 SMP/REC를 수정합니다.
6. `시나리오 비교`에서 가격 하락, 설치비 증가, 금리 상승에 대한 내성을 봅니다.
7. `리스크 점검`에서 손익분기 발전가격과 설치비 한계를 확인합니다.
8. `요약 보고서 PDF 받기`로 핵심 결과를 저장합니다.
9. 공유 URL을 복사해 농가, 상담자, 시공사, 지자체 담당자가 같은 입력값을 기준으로 논의합니다.

## 9. 공유 URL 사용법

사이드바 하단의 `결과 공유` 링크를 복사하면 현재 입력값이 URL에 저장됩니다.

예를 들어 URL에는 다음과 같은 값이 포함됩니다.

| 키 | 의미 |
|---|---|
| m | 입력 모드, simple 또는 expert |
| a | 농지 면적 |
| c | 시설 용량 |
| h | 1일 평균 발전시간 |
| b | 총 사업비 |
| e | 자기자본 비율 |
| l | 융자 조건 |
| r | 금리 |
| t | 발전가격 트랙 |
| s | SMP |
| x | REC |
| w | REC 가중치 |
| p | PPA 단가 |

공유 URL을 열면 같은 입력값으로 결과가 재현됩니다. 세부 파라미터가 들어간 기존 링크는 전문가 모드로 열리도록 호환성을 유지했습니다.
